# Uber Duck Script
## A CLI for uberduck.ai


## Requirements
ffmpeg
unix-based system
curl

## Usage

```sh
chmod +x uberduck.sh
./uberduck.sh [file here.txt] [voice_name] (found in sounds.txt)
./uberduck.sh poem.txt eminem
```
the more you divide the text into separate lines the better the results will be
